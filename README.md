# Portfolio Website

モダンでレスポンシブなポートフォリオサイトです。

## 概要

このポートフォリオサイトは、HTML、CSS、JavaScriptを使用して構築されたシングルページアプリケーションです。モダンなデザインとスムーズなアニメーションを特徴としています。

## 特徴

- ✨ モダンで洗練されたデザイン
- 📱 完全レスポンシブ対応
- 🎨 美しいアニメーション効果
- ⚡ 軽量で高速
- 🎯 スムーズなスクロールナビゲーション

## セクション

- **ホーム**: 自己紹介とヒーローセクション
- **About**: 詳細な自己紹介
- **スキル**: 技術スキルの紹介
- **プロジェクト**: 過去のプロジェクト一覧
- **お問い合わせ**: 連絡先フォーム

## 技術スタック

- HTML5
- CSS3 (カスタムプロパティ、Flexbox、Grid)
- Vanilla JavaScript
- Google Fonts (Inter)

## セットアップ

1. リポジトリをクローン:
```bash
git clone https://github.com/FujihitoWatanabe/portfolio.git
cd portfolio
```

2. ブラウザで `index.html` を開く

または、ローカルサーバーを起動:
```bash
# Python 3の場合
python -m http.server 8000

# Node.jsの場合
npx serve
```

## カスタマイズ

### 個人情報の更新

`index.html` ファイル内の以下のセクションを編集してください:

- 連絡先情報（お問い合わせセクション）
- プロジェクト情報
- スキルリスト
- 統計情報

### スタイルの変更

`styles.css` ファイルのCSS変数を編集することで、カラーテーマを簡単に変更できます:

```css
:root {
    --primary-color: #6366f1;
    --secondary-color: #8b5cf6;
    /* ... */
}
```

## GitHub Pagesでのデプロイ

1. リポジトリの Settings に移動
2. Pages セクションを開く
3. Source を "main" ブランチに設定
4. 保存後、数分でサイトが公開されます

URL: `https://fujihitowatanabe.github.io/portfolio/`
